== URL deprecation blocklist ==
This page lists domains and URL patterns that are considered non-qualifying for notability purposes on Wikidata. It is intended to be used by bots and tools to automatically apply [[Help:Ranks|deprecated rank]] to URL-type statements where the domain is known to be a self-created profile platform, press release distributor, wiki farm, or similar non-independent source.

=== How to use this page ===
Each section corresponds to a deprecation reason. Each table row lists:
* '''Pattern''': the domain or URL prefix to match
* '''Match type''': either '''prefix''' (match from start of URL) or '''regex''' (full regular expression)
* '''Example URL''': a representative example
* '''Notes''': any relevant context or edge cases
* '''Added''': date added to the list

Bots should match URLs against the pattern column using the specified match type, apply deprecated rank, and set "reason for deprecated rank" to the item corresponding to the section heading.

For '''prefix''' matching, normalize URLs by stripping the <code>https://</code>, <code>http://</code> and <code>www.</code> prefix before matching.

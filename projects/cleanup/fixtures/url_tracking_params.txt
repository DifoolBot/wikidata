This page lists URL query parameters that are automatically stripped from URLs by [[User:Difool/WikidataCleanup|WikidataCleanup]]. The data is cached locally for seven days.
== Format ==
The table has three columns:

* '''Hostname''' – the domain to match. A plain hostname such as <code>twitter.com</code> also matches subdomains like <code>fr.twitter.com</code> or <code>m.twitter.com</code>. A hostname starting with a dot such as <code>.scholar.google</code> matches any country-code variant like <code>scholar.google.com.au</code> or <code>scholar.google.co.uk</code>.
* '''Mode''' – either <code>always</code> (the parameter is always removed) or <code>recognition</code> (the parameter is only removed when the URL is being matched against a known external ID pattern).
* '''Parameters''' – a comma-separated list of URL query parameter names to remove.

== Adding entries ==
Before adding a new entry, verify that the parameter serves no purpose other than tracking or personalisation and carries no information relevant to identifying the source. Parameters that affect which content is displayed (such as a user ID or article identifier) should not be added.
